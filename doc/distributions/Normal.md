Normal distribution
===================
Normal(Object **mean**, Object **sd**)
--------------------------------------

The normal probability distribution.

### Parameters

- Object **mean** - the mean of the distribution.
- Object **sd** - the standard deviation of the distribution.

### Return type

- Object
